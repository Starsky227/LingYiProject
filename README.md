### 📋 系统要求

- **操作系统**: Windows 11 （10应该也行？）
- **Python**: 3.10+ (推荐 3.13)
- **内存**: 建议 4GB+ （需要跑neo4j）
- **存储**: 建议 5GB+ （取决于你需要存多少数据，不过部署要求不高）
