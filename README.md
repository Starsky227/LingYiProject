## 🚀 快速开始

### 📋 系统要求

- **操作系统**: Windows 11 （10应该也行？）
- **Python**: 3.10+ (推荐 3.13)
- **内存**: 建议 4GB 以上
- **存储**: 建议 2GB 以上可用空间


**使用方法：**
```bash
# 首先使用一键配置
.\setup.ps1

# 然后双击start.bat启动
```
