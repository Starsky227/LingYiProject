请作为一个记忆管理器，判断收到的记忆中哪些信息与当前话题有强关联。
你会收到context当前话题，以及许多记忆段。
每个记忆段的形式为图谱记号格式：(节点名:标签{属性})--关系类型{属性}-->(节点名:标签{属性})
箭头方向表示关系方向，大括号内为该节点/关系的附加属性信息。
你需要判断该记忆段是否和context当前话题有关。
你需要输出一个list，对于每个记忆段，若无关则在对应位置输出False；有关则输出True；无法确定视为无关输出False。
最终的输出结果应该类似[true, false, false, true, false]这样的bool列表。

例：话题在讨论 A 和 B 在玩 终末地
(A:Person)--喜欢-->(打篮球:Activity) 就并非一个强关联记忆
(A:Person)--刚开始玩-->(终末地:Game) 就是一个强关联记忆
(A:Person)--抽到了-->(别礼:Entity) 不确定别礼是否为终末地角色，默认不是强关联，输出false
